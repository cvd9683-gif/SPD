# SPD Website Redesign

UX case study and presentation deck for The Society of Publication Designers website redesign.

## Files

- **`spd-case-study.html`** — Long-form UX case study. Process embedded throughout (problem & research setup, insights with research artifacts, focus & direction shift, structure & system design, key flows, reflection). Includes sticky navigation, clickable wireframe modal, and a print stylesheet for PDF export.
- **`spd-presentation.html`** — 14-slide presentation deck. Scroll-snap navigation, arrow keys to advance, full-bleed accent slides for key moments. Print export at 16:9 letterbox.

Both files are self-contained HTML with embedded CSS/JS — open in a browser, no build step.

## Status

In progress. Quantitative outcomes will be added once the redesigned site has been live long enough to measure against the existing baseline.
